## My personal dotfiles

Stack:
 - Terminal: **Urxvt**
 - Window manager: **i3wm**
 - Shell: **zsh**

## Dependencies
- [i3blocks-contrib](https://github.com/vivien/i3blocks-contrib)
## Deploy using stow
``` shell
  stow <package-name>
```
